# Ankes-Sales-Analysis
This project is a multi-page interactive Power BI dashboard analysing retail sales data across 49 U.S. states from 2021 to 2024. It covers over 32,000 units sold across three product categories Technology, Furniture, and Office Supplies and tracks key business metrics including revenue, profitability, loss exposure, and customer behaviour.

## 📌 Project Overview

This dashboard was built to provide a comprehensive view of business sales performance — identifying revenue trends, profitability by category and location, loss-generating segments, and high-value customers. It is designed to support data-driven decision-making for sales and management teams.

## 📋 Dashboard Pages

The report contains **8 interactive pages**, each focusing on a different analytical dimension:

| Page | Description |
|---|---|
| **Overview** | High-level KPIs, profit/loss by year, product data table |
| **Sales Analysis** | Sales by year, month, category, and sub-category |
| **Profit Analysis** | Profit breakdown by sub-category, month, year, and state |
| **Loss Analysis** | Loss by sub-category, month, state, and year |
| **Location Analysis** | Sales, profit, and loss mapped across U.S. states |
| **Product Analysis** | Top products by revenue, profit, loss, and quantity |
| **Data Table** | Full granular product-level data with all metrics |
| **Customer Analysis** | Sales, profit, loss, and quantity by customer |
